# git lesson

## リモートリポジトリとローカルリポジトリとはそれぞれなんでしょうか？
- リモートリポジトリ
  - 複数人で共有するためのリポジトリ
- ローカルリポジトリ
  - 開発者が自分のpcに取り入れたリポジトリ

## プッシュとマージの違いは何でしょうか？
- プッシュ
  - ローカルからリモートにコミットした内容を反映させる
- マージ
  - ブランチの変更内容を別のブランチに反映させる

## コミットとプッシュの違い
- コミット
  - 操作ミスなどによりがデータが消えてしまった際に修復ができない
- プッシュ
  - データが消えてしまってもリモートにプッシュした内容をローカルに反映できるので消えた地点から作業を再開できる


## コミットのメッセージはどのように書いてあげるのが最適でしょうか？
保存した内容が他者にも理解できるように書く

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？
- ローカル
  - レビュー、承認の必要がないためバグがあっても発見できない
- プルリクエスト
  - レビュー、承認の必要がありバグがあっても事前に発見できる

## コンフリクトを起こしてしまった場合、どう対処すべきですか？
他者が書いたソースコードの内容を把握する必要があるため、ソースコードを書いた人と相談しながら作業を進める
